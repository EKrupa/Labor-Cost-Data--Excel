# Labor Cost Analysis – Excel Project

## Overview
This project analyzes labor cost data using Microsoft Excel to identify cost trends, department-level spending patterns, and workforce efficiency insights. Raw labor data was cleaned and structured, then summarized using PivotTables for dynamic analysis.

## Tools Used
- Microsoft Excel
- PivotTables
- Data Cleaning & Structuring
- Calculated Fields
- Basic Cost Analysis

## Key Analysis
- Aggregated total labor costs by department and time period  
- Compared labor hours vs. total labor spend  
- Identified high-cost areas and cost distribution patterns  
- Created dynamic PivotTables for interactive reporting  

## Business Value
This analysis demonstrates how Excel can transform raw labor data into actionable insights that support budgeting, cost control, and operational decision-making.
